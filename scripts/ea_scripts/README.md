# LiQiyu Strategy V8.4 Sniper (XAUUSD EA)

> **"Sniper Mode: Zero Latency, Maximum Precision."**
> A robust breakout strategy upgraded with Pending Orders for millisecond execution.

## 核心升级：狙击手模式 (V8.4 Sniper)
相比于 V8.3，本版本进行了重大的底层升级：
*   **不再等待**：放弃了“价格到了再开单”的市价单模式 (Market Execution)。
*   **主动出击**：采用 **挂单 (Pending Orders)** 模式。
    *   EA 会在每一根 M3 K线开始时，直接在计算好的突破点位挂上 `BUY STOP` 或 `SELL STOP`。
    *   **0 延迟**：价格一碰，交易所服务器直接成交，完全消除了网络延迟和计算耗时。
    *   **自动撤单**：如果 3 分钟内（M3周期）没有突破，EA 会自动撤单，等待下一个机会。

## 1. 核心逻辑 (Core Logic)
仅当以下 7 个条件同时满足时，EA 会部署“狙击陷阱” (挂单)：

1.  **[M30] 大势确认**: M30 收盘价 > M30 EMA(34)。(顺势)
2.  **[M12] 趋势确认**: M12 收盘价 > M12 EMA(34)。(顺势)
3.  **[ADX] 动能强度**: M30 ADX(14) > **20**。(拒绝死鱼盘)
4.  **[Vol] 极度缩量**: M3 成交量 < M3 均量的 **0.60**。(蓄势待发)
5.  **[RSI] 多周期共振**: M3 不超买/超卖，M12 在趋势区。(空间确认)
6.  **[Mom] 动能惯性**: M12 收盘价抬升/下降。(拒绝逆势接刀)
7.  **[Brk] 价格突破**: 突破 M3 高低点 + **30点** 缓冲。(精确打击)

## 2. 风控机制 (Risk Management)
*   **止损 (SL)**: 基于 ATR 的硬止损 (`ATR * 3.5`)。
*   **止盈 (TP)**: 移动止损 (`Trailing Stop`)。浮盈 > 1.5倍 ATR 后启动，锁定利润。
*   **资金**: 固定手数 (`0.01` 起步)。

## 3. 安装与使用 (Install)
1.  将 `.mq4` 文件放入 MT4 的 `MQL4/Experts` 目录并编译。
2.  将 EA 拖入 **M1 (1分钟)** 图表。
3.  **重要**：你会看到图表上出现彩色的虚线 (`buy stop` / `sell stop`)。
    *   这是 EA 正在工作，它在“瞄准”。
    *   **不要手动删除这些挂单**，EA 会自动管理它们。

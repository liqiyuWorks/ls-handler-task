## 1. 核心逻辑 (Core Logic)
本策略放弃高频交易，专注于**高确定性机会**。只有当以下 7 个条件同时满足时，才会开多/开空：

1.  **[M30] 大势确认**: M30 收盘价 > M30 EMA(34)。(顺势)
2.  **[M12] 趋势确认**: M12 收盘价 > M12 EMA(34)。(顺势)
3.  **[ADX] 动能强度**: M30 ADX(14) > **20**。(拒绝死鱼盘)
4.  **[Vol] 极度缩量**: M3 成交量 < M3 均量的 **0.60**。(暴风雨前的宁静)
5.  **[RSI] 多周期共振**: M3 不超买/超卖，M12 在趋势区。(空间确认)
6.  **[Mom] 动能惯性**: M12 收盘价抬升/下降。(拒绝逆势接刀)
7.  **[Brk] 价格突破**: 实质突破 M3 高低点 + **30点** 缓冲。(不见兔子不撒鹰)

---

## 2. 风控机制 (Risk Management)
*   **止损 (SL)**: 基于 ATR 的硬止损 (`ATR * 3.5`)。每一单都有保护。
*   **止盈 (TP)**: 移动止损 (`Trailing Stop`)。浮盈 > 1.5倍 ATR 后启动，让利润奔跑。
*   **资金**: 固定手数 (`0.01` 起步)，无马丁，无网格。

## 3. 参数配置指南 (Configuration)
不想改代码？直接在 MT4 输入参数中调整：

| 参数 | 稳健版 (V8.3 Default) | 激进版 (Test Mode) | 说明 |
| :--- | :--- | :--- | :--- |
| **Vol_Squeeze_F** | `0.60` | `0.80` 或 `1.0` | 缩量因子。越小越难进。0.8 适合测试。 |
| **ADX_Min_Level** | `20` | `15` 或 `0` | 趋势门槛。设为 0 则无视趋势强弱。 |
| **Use_M30_Filter**| `true` | `false` | 是否看大周期面子。`false` 信号更多但风险大。 |
| **Use_RSI_Filter**| `true` | `false` | 是否过滤超买超卖。 |

## 4. 安装与使用
1.  将 `.mq4` 文件放入 MT4 的 `MQL4/Experts` 目录。
2.  在 MT4 中点击“编译”。
3.  将 EA 拖入 **M1 (1分钟)** 图表。(EA 内部会自动合成 M3/M12 数据，无需挂其他周期)。
4.  确保“允许实时自动交易”已勾选。
